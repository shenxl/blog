金山智慧校园配置助手隐私政策

最后更新日期：2025年4月28日

引言

金山智慧校园配置助手是一款设计用于帮助用户自动配置金山智慧校园平台的 Chrome 扩展程序。本隐私政策旨在明确说明本扩展如何收集、使用和保护用户数据，以及我们对用户隐私的尊重和保护措施。

收集的信息

本扩展仅收集以下类型的用户数据，且仅用于实现其功能：

  • 认证信息：包括用户在 WPS 智慧校园平台（*.wps.cn 和 *.kdocs.cn 域名下）的 Cookie 和 CSRF 令牌。这些信息仅用于进行自动化配置操作。
  • 任务执行状态：记录用户启动的自动化配置任务的执行状态和日志。
  • 配置选择：用户在扩展中选择的配置选项，如模板类型（基教/高校）。
重要提示： 所有收集的数据仅存储在您的本地设备上，不会上传至任何服务器。

数据使用

收集的数据仅用于以下目的：

  • 执行自动化配置：使用用户的认证信息向 WPS 智慧校园平台发送请求，执行应用上架、分组设置、工作台配置等自动化任务。
  • 状态恢复：使任务可以在中断后继续执行，无需重新开始。
  • 用户体验优化：基于用户的选择记忆上次的配置，避免重复输入。

数据存储

所有收集的数据都仅存储在用户本地设备上：

  • 扩展使用 Chrome 提供的 storage.local API 或 IndexedDB 进行数据存储，这些数据不会上传至任何云端服务器。
  • 认证信息（Cookie 和 CSRF 令牌）仅保存在本地，并在与 WPS 服务器通信时使用。

数据共享

本扩展不会与任何第三方共享用户数据：

  • 收集的认证信息仅用于与 WPS 官方服务器（*.wps.cn 和 *.kdocs.cn）通信，这与用户直接在浏览器中访问这些网站时的行为一致。
  • 不会将任何用户数据发送到非 WPS 官方的第三方服务器。

权限使用说明

本扩展请求以下权限，每项权限都是实现功能所必需的：

activeTab：用于与用户当前访问的 WPS 相关网站交互。
storage：用于在本地存储用户认证信息和配置数据。
scripting：用于在 WPS 网站上执行自动化操作的脚本。
sidePanel：用于提供扩展的用户界面。
cookies：用于读取 WPS 相关网站的 Cookie，获取必要的认证信息。
webRequest：用于向 WPS 服务器发送经过认证的网络请求。
主机权限 (*://*.wps.cn/*, *://*.kdocs.cn/*)：仅用于与这些域名的服务器通信，执行自动化配置任务。

数据安全

我们采取以下措施保护用户数据：

  • 仅收集执行任务所必需的最小数据集。
  • 所有数据仅保存在用户本地设备上，不会上传至云端。
  • 在控制台日志中隐藏敏感信息（如 Cookie 值、CSRF 令牌）。
  • 不会处理与扩展功能无关的用户数据。

数据删除

如果您想删除扩展存储的所有数据：

  • 在 Chrome 浏览器中卸载扩展，这将自动清除所有由扩展存储的数据。
  • 您也可以通过清除浏览器的站点数据（特别是关于 *.wps.cn 和 *.kdocs.cn 域名的数据）来移除相关 Cookie。

政策更新

本隐私政策可能会随着扩展功能的更新而变更。我们会在扩展更新时通知用户任何重大变更，并在 Chrome Web Store 的扩展页面保持政策的最新版本。

联系方式

如果您对本隐私政策有任何疑问或顾虑，请通过 Chrome Web Store 扩展页面上提供的联系方式与我们联系。
